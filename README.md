# BINF-5507-Suicide-Detection

### Abstract
Project uses MLP (Multi-Layer Perceptron) to detect probable suicide messages based on social media posts, distinguishing Reddit posts that indicate suicide and non-suicide. The MLP is compared against Logistic Regression, SVM, and Bernoulli Naive Bayes classifiers.

---

## Datasets

| Role | File | Size | Balance |
|---|---|---|---|
| Training | `datasets/sample_100_tokens1.csv` | 92 posts | 52 suicide / 40 non-suicide |
| Testing | `datasets/sample_20_tokens.csv` | 19 posts | 13 suicide / 6 non-suicide |
| Full training source | `datasets/Suicide_Detection.csv` | 232,074 posts | perfectly balanced (50/50) |
| Full test source | `datasets/reddit_depression_suicidewatch_tokens1.csv` | 19,257 posts | r/SuicideWatch vs r/depression |

**Feature representation:** Each post is tokenized using `bert-base-cased` (BERT). Token IDs are stored as integer sequences and zero-padded to a fixed length (1964 positions — the longest post in the dataset), forming a `(n_posts × 1964)` feature matrix fed into all classifiers.

**Label mapping:**
- `suicide` → 1
- `non-suicide` → 0
- For the Reddit test set: `r/SuicideWatch` → suicide, `r/depression` → non-suicide

## Section 1.5 — Classifier Comparison

All four classifiers were trained on the same feature matrix and evaluated on the same test set.

| Classifier | Accuracy | Precision | Sensitivity | Specificity | F1 | AUROC |
|---|---|---|---|---|---|---|
| **MLP** | **94.7%** | **1.000** | 0.923 | **1.000** | **0.960** | **1.000** |
| Logistic Regression | 89.5% | 0.923 | 0.923 | 0.833 | 0.923 | 0.872 |
| SVM | 73.7% | 0.750 | 0.923 | 0.333 | 0.828 | 0.910 |
| Bernoulli NB | 68.4% | 0.889 | 0.615 | 0.833 | 0.727 | 0.737 |

**Key findings:**
- **MLP** achieves the best performance across all metrics. Perfect AUROC (1.0) indicates complete class separability on this test set.
- **SVM** has the worst specificity (0.333) — it falsely flagged 2 out of 6 non-suicide posts as suicide, even though it caught all actual suicide posts.
- **Bernoulli NB** has the lowest sensitivity (0.615) — it missed 5 out of 13 real suicide posts, the most dangerous failure mode in a clinical context.
- **Logistic Regression** is the strongest linear baseline, competitive across all metrics with a much simpler model.
- **Sensitivity is the most important metric** in this task: missing a suicide post (false negative) carries far greater risk than a false alarm (false positive).

## How to Run

**1. Clone the repo**
```bash
git clone https://github.com/AlizeDeMatas/BINF-5507.git
cd BINF-5507
```

**2. Set up the environment**
```bash
conda env create -f environment.yml
conda activate suicide-detection
```
Or with pip:
```bash
pip install -r requirements.txt
```

**3. Download NLTK data**
```bash
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords'); nltk.download('punkt_tab')"
```

**4. Tokenize the data**

Open `2_bert_tokenize.py` and set `name = 'sample_100'` on line 8, then run:
```bash
python 2_bert_tokenize.py
```
Then change `name = 'sample_20'` and run again:
```bash
python 2_bert_tokenize.py
```
This generates `sample_100_tokens.csv` and `sample_20_tokens.csv`.

**5. Run the MLP classifier only**
```bash
python mlp.py
```
Results print to terminal and a confusion matrix is saved as `mlp_confusion_matrix.png`.

**6. Run the full assignment (Sections 1.4 & 1.5)**
```bash
python assignment_1_4_and_1_5.py
```
Runs MLP and all comparison classifiers, prints a full results table, and saves 4 plot PNGs to the project folder.
